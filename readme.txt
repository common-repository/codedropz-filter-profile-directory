A simple profile directory plugin and easy to manage using custom postype with custom filter.

== Description ==

Profile Directory Filter is a simple plugin that will create profile information and displayed as directories with built-in Alphabetical Letters filter, Ajax and custom popup.

== Features ==

* Easy to customize and use
* Filter alphabetically by Lastname or FirstName
* Sort by LastName or FirstName
* Support Ajax and Custom Modal/Popup
* Create Profie Information
* Fully Responsive & Optimized for all Devices
* Build using custom post type
* Has an option to use bootstrap as Template
* Support upto 6 column grid (an option in the backend)
* Can adjust/set image size
* No additional coding required

== Installation ==

To install this plugin see below:

1. Upload the plugin files to the `/wp-content/plugins/codedropz-profile-directory-filter` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Profile Directory > Settings screen to configure the plugin
4. For basic usage and configuration please visit [documentation](http://codedropz.com/documentation/).
5. Demo [here](http://codedropz.com).
